## Produto Cartesiano

<div class="regular">

**Exemplo**

```c
struct data {
    int dia, mes, ano;
};
```

**Inicialização**
```c
struct data d = { 7, 9, 1999 };
```

**Acesso**
```c
printf("%d/%d/%d\n", d.dia, d.mes, d.ano);
```

**Cardinalidade**
```
#INTEGER x #INTEGER x #INTEGER
```
